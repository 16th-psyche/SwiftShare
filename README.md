# SwiftShare

A lightweight macOS menu-bar utility for **URL shortening** and **temporary file
sharing** — no Dock icon, no background daemon, no account.

It ships in two forms:

1. **Menu-bar app** (`SwiftShare.app`) — a one-click icon in the top bar with a popover for
   shortening URLs and drag-and-drop uploads with a live progress bar.
2. **Quick Action scripts** — zero-install zsh scripts driven from the right-click menu,
   for those who'd rather not build a binary.

Everything relies only on tools already shipped with macOS (`swiftc`, `zsh`, `curl`, `zip`).

---

## Menu-bar app (recommended)

A single-file SwiftUI/AppKit app compiled with `swiftc` — **no Xcode project required**
(Command Line Tools are enough).

```sh
./build.sh           # compiles app/main.swift and bundles ./SwiftShare.app
open SwiftShare.app   # the paper-plane icon appears in the menu bar
```

Click the menu-bar icon to open the popover. Three tabs:

- **Shorten** — type/paste a URL or hit **Paste**, then **Shorten** (or press Return). The
  short link is shown, copied to your clipboard, and added to History. Uses direct-redirect
  shorteners (**is.gd**, falling back to **da.gd**) — no interstitial/preview page.
- **Upload** — drag files/folders onto the drop zone (or **Choose**). They stage into a list;
  click **Upload**. A **single file** uploads as-is, a **folder** is zipped, and **multiple
  items are bundled into one `.zip` with one shared link**. A progress bar tracks the upload;
  the result is shown, copied, and labeled with its retention window. The panel grows with the
  number of files and only scrolls past ~70% of screen height.
- **History** — your recent links (persisted), each with copy/open buttons.

To launch at login: **System Settings → General → Login Items → +** and add `SwiftShare.app`.

> The app is ad-hoc signed for local use. Moving it to another Mac re-quarantines it —
> just rebuild there with `./build.sh`.

## Quick Action scripts (zero-install alternative)

Two zsh scripts that lean only on pre-installed macOS tools:

- [scripts/swiftshare-shorten.sh](scripts/swiftshare-shorten.sh) — URL shortener (is.gd → da.gd).
- [scripts/swiftshare-upload.sh](scripts/swiftshare-upload.sh) — file/folder uploader with a
  host fallback chain.

Both take their input as the first argument (`$1`); the shortener also reads stdin.

```sh
chmod +x scripts/*.sh

# Shorten
./scripts/swiftshare-shorten.sh "https://example.com/a/very/long/path?with=params"
pbpaste   # -> https://is.gd/XXXXXX

# Upload a file, or a folder (zipped automatically)
./scripts/swiftshare-upload.sh ~/Desktop/report.pdf
./scripts/swiftshare-upload.sh ~/Desktop/build-assets
```

### Wiring them as right-click Quick Actions

Create one Quick Action per script in **Automator** (`New → Quick Action`):

- **Shorten Link** — *receives* **text** in **any application** → **Run Shell Script**
  (Shell `/bin/zsh`, Pass input **as arguments**):
  ```sh
  "/path/to/SwiftShare/scripts/swiftshare-shorten.sh" "$1"
  ```
- **Upload & Share** — *receives* **files or folders** in **Finder** → **Run Shell Script**
  (same settings):
  ```sh
  "/path/to/SwiftShare/scripts/swiftshare-upload.sh" "$1"
  ```

Replace `/path/to/SwiftShare` with this repo's location. They then appear under right-click →
**Quick Actions**; assign keyboard shortcuts in **System Settings → Keyboard → Keyboard
Shortcuts → Services**.

## Upload hosts (fallback order)

The uploader tries each host in order and stops at the first that returns a usable link.

| Host | Max payload | Retention |
| --- | --- | --- |
| litterbox.catbox.moe | 200 MB | 72 hours (primary) |
| 0x0.st | 512 MB | 30 days |

> **Ephemeral by design.** These are zero-login scratch hosts with automatic deletion.
> Do **not** use SwiftShare for long-term archives. Files over the largest host cap are
> flagged before upload.

## Project structure

```
SwiftShare/
├── app/main.swift          # the menu-bar app (SwiftUI + AppKit, single file)
├── icon/GenerateIcon.swift # offline generator for the app icon (run by build.sh)
├── scripts/                # zsh Quick Action scripts (shorten / upload)
├── build.sh                # compiles + bundles SwiftShare.app
└── README.md
```

Build artifacts (`SwiftShare.app/`, `icon/AppIcon.icns`, `icon/AppIcon.iconset/`) are
generated by `build.sh` and are git-ignored.

## Requirements

- macOS 13 or later
- Xcode Command Line Tools (`xcode-select --install`) — provides `swiftc`, `iconutil`
